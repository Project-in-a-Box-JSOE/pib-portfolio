Imagine if you needed to print "I'm Alive!" 1000 times. Without Loops, that would be 1000 lines of code. With a loop, it would look like this: ``` for (_____) { // Code Block Here } and ``` The for statement will contain 3 parameters that determine how many times to loop for. The following is a concrete example: ``` for (var i = 0; i < 1000; i++){ console.log("I'm Alive! Its the" + i + "nth print"); } ``` Breaking this down, in javascript: First, we create a counter and set it to 0. Second, we state the *Condition* that the counter i must be less than 1000. Third, we have the increment, where you set how much you update i by each loop (we chose 1) By putting i++, we are telling the program to increment i by 1 for each iteration of the loop. In Summary, this should loop 1000 times, starting at i = 0, and then ending at i = 999, where at the end it will reach 1000 and not run the code block within the sections Now's a good time to pop open exercise #1 and attempt the exercise! <br/> (Highly recommended breakpoint to consolidate what you've learned) Now to Dive into code.
